Mac OS X versions (builds) included with Intel-based Macs

  • Last Modified: December 23, 2008
  • Article: HT1159
  • Old Article: 303315

Summary

Learn which versions of Mac OS X ship with Intel-based Macs. If you have a PowerPC-based Mac, see this article instead.

Important: Do not use a Mac OS version earlier than the one included with the computer.

Notes

  1. Some computers shipped with different Mac OS X versions at different times. When a computer is not bundled with a later version, a "-" appears in the "Later Mac OS included" column.
  2. The build information is available in the About This Mac window.
  3. Some Macs may include a slightly later build version (but same numbered version) of Mac OS X on the Install or Restore discs than was preinstalled on the computer.
