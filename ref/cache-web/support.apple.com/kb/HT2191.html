Mac OS: Versions, builds included with PowerPC Macs (since 1998)

  • Last Modified: July 16, 2008
  • Article: HT2191
  • Old Article: 25517

Summary

Learn the version(s) of Mac OS included with PowerPC Macs produced since 01 Jan 1998. If you're looking for information about Intel Macs produced since 2006, see this article.

Important: You cannot normally use a Mac OS version earlier than the one included with the computer. See "Don't install older versions of Mac OS than what comes with your computer," as well as the related documents mentioned below.

Products Affected

Power Mac Server, Xserve, iBook, MacBook, PowerBook, Power Mac, Mac mini, iMac, eMac

  1. Some computers shipped with different Mac OS versions at different times. When a computer was not bundled with a later version, a "-" appears in the "Later Mac OS included" column.
  2. The build information is available in the About This Mac window.
  3. The parenthetical product descriptions (Summer 2000) and (Summer 2001) refer to the summer of earth's Northern Hemisphere.
  4. Also known as iBook (Early 2003).
  5. (*) Some, or all, of this type of computer start in Mac OS X only. For more information, see "Macintosh: Some Computers Only Start Up in Mac OS X". Additional information:

    • iMac (Flat Panel) with serial numbers of xx303xxxxxx or later start only in Mac OS X
    • eMac (ATI Graphics) with SuperDrive start only in Mac OS X.
  6. Some computers may include a slightly newer build (but same numbered version) of Mac OS X on the Install or Restore discs than was preinstalled on the hard drive. This is normal.
  7. Some versions of computer-specific software can only be used for NetBoot or System Images for the computer model with which they were bundled. This includes these Power Mac G5 computer-specific builds: 6S74, 6S75, 6S80, 6S90.
